# Job Finder: Парсер и уведомитель вакансий VK Education

[![Language: RU](https://img.shields.io/badge/Language-RU-blue.svg)](README.ru.md)
[![Language: EN](https://img.shields.io/badge/Language-EN-green.svg)](README.md)

Автоматизированная событийно-ориентированная система для отслеживания стажировок и junior-вакансий на портале **VK Education** с отправкой мгновенных уведомлений в Telegram.

## 🛠 Стек технологий
* **Go** (Сервис обработки данных, Telegram-бот)
* **Python** (Парсер на базе Playwright)
* **Apache Kafka / Redpanda** (Брокер сообщений)
* **PostgreSQL** (Реляционная база данных)
* **gRPC** (Межсервисное взаимодействие)
* **Docker & Docker Compose** (Контейнеризация)

## 📐 Обзор архитектуры
Система спроектирована по микросервисному принципу:
1. **Парсер (Python):** Собирает вакансии с сайта через Playwright и отправляет их потоком в Kafka.
2. **Обработчик (Go):** Получает вакансии из очереди, дедуплицирует по URL, нормализует данные и сохраняет в PostgreSQL.
3. **Telegram-бот (Go):** Оповещает пользователя о новых вакансиях и обрабатывает интерактивные команды (например, `/parse_vacancy` через gRPC).
